# NetManTrigger

A utility to enumerate Windows network interfaces using the `INetConnectionManager` COM interface.

## Credits & References

This tool is based on the following resources:

- **DLL Hijacking research** by itm4n:  
  https://itm4n.github.io/windows-server-netman-dll-hijacking/

  https://stackoverflow.com/questions/5917304/how-do-i-detect-a-disabled-network-interface-connection-from-a-windows-applicati/5942359#5942359
